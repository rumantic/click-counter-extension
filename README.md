# Click Counter - Расширение для Chrome

Расширение для подсчета кликов мышкой пользователя в браузере Chrome.

## Возможности

- ✅ Подсчет общего количества кликов
- ✅ Раздельный подсчет кликов левой, правой и средней кнопки мыши
- ✅ Сохранение статистики между сеансами
- ✅ Возможность сброса счетчика
- ✅ Красивый интерфейс popup

## Установка

1. Откройте Chrome и перейдите по адресу `chrome://extensions/`
2. Включите "Режим разработчика" в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку с расширением `click-counter-extension`
5. Расширение установлено! Иконка появится на панели инструментов

## Использование

1. После установки расширение автоматически начнет считать клики на всех веб-страницах
2. Кликните по иконке расширения, чтобы увидеть статистику
3. Нажмите кнопку "Сбросить счетчик", чтобы обнулить все значения

## Примечание

Для полноценной работы расширения нужно добавить иконки в папку `icons/`:
- icon16.png (16x16 пикселей)
- icon48.png (48x48 пикселей)
- icon128.png (128x128 пикселей)

Вы можете создать простые иконки с изображением курсора или счетчика.
