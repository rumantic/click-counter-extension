# Политика конфиденциальности MouseStat

**Дата вступления в силу: 8 января 2026**

## Сбор данных

Расширение MouseStat **НЕ собирает, НЕ передает и НЕ хранит** никакие персональные данные пользователей.

## Хранение данных

Расширение сохраняет только локальную статистику использования мыши:
- Общее количество кликов
- Количество кликов левой кнопки мыши
- Количество кликов правой кнопки мыши
- Количество кликов средней кнопки мыши
- Пройденная курсором дистанция (в пикселях)
- Дистанция прокрутки колесиком (в пикселях)

Все данные хранятся **локально в браузере пользователя** через Chrome Storage API и **никогда не передаются** на внешние серверы.

## Разрешения

Расширение запрашивает следующие разрешения:

### storage
Используется для локального сохранения статистики кликов и дистанции. Данные хранятся только на вашем устройстве.

## Передача данных третьим лицам

Расширение **НЕ передает** никакие данные третьим лицам. Расширение работает полностью автономно и не требует подключения к интернету.

## Безопасность

- Все данные остаются на устройстве пользователя
- Данные могут быть удалены в любой момент через настройки расширения или кнопку "Сбросить счетчик"
- Удаление расширения автоматически удаляет все сохраненные данные

## Что мы НЕ делаем

- ❌ НЕ собираем личную информацию
- ❌ НЕ отслеживаем историю посещений
- ❌ НЕ используем аналитику
- ❌ НЕ показываем рекламу
- ❌ НЕ передаем данные на серверы
- ❌ НЕ используем куки
- ❌ НЕ требуем регистрации
